# fintech-homepage

Personal homepage (FinTech PhD) — learn by doing, learn in public.

## Local preview

Just open `index.html` in your browser.

## Deploy to GitHub Pages (recommended)

1. Create a GitHub repo named `fintech-homepage` (or any name you like).
2. Upload/push all files in this folder.
3. In GitHub: **Settings → Pages**
   - **Build and deployment**: Deploy from a branch
   - **Branch**: `main` / `(root)`
4. Visit your Pages URL.

## Content structure

- `content/fintech.md`: FIN notes (PQE lecture notes, etc.)
- `content/tech.md`: ML/DL/GL/LLM/Agent notes
- `content/demos.md`: demos portfolio
- `content/interns.md`: internship pitch + contact
- `content/discussions.md`: discussion prompts + links
