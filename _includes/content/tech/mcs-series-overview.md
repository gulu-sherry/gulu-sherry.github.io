# 蒙特卡洛模拟（Monte Carlo Simulation）课程笔记
> 系列入口页：用于持续整理这门课的每个 Topic 笔记、公式、直觉解释与常见误区。

## 课程定位

这是一套面向网页阅读的课程笔记系列，主题是 **蒙特卡洛模拟（Monte Carlo Simulation）**。  
整门课后续按 **4 个 Topic = 4 篇教程笔记** 的方式持续整理，而不再按零散 lecture 拆分。每篇笔记都会重点回答：

- 这一部分讲了什么
- 为什么重要
- 数学上怎么表达
- 实际上怎么理解
- 常见误区是什么

## 更新方式

当你提供新的 Topic 幻灯片、讲义、文稿或课程链接后，这个系列会按统一模式增量更新：

1. 在 `/_includes/content/tech/` 中新增对应 Topic 正文
2. 在 `/_pages/` 中新增对应 Topic 页面并配置 permalink
3. 在 `/_data/monte_carlo_course.yml` 中登记 Topic 信息
4. 在主页 `Tech` 区域自动显示对应 Topic 入口

## 计划中的单篇结构

每个 Topic 默认按下面的模板整理：

| 模块 | 内容 |
| --- | --- |
| 标题 | Topic 标题、主题、资料来源 |
| 核心问题 | 这个 Topic 想回答什么问题 |
| 概念拆解 | 关键概念、术语、相互关系 |
| 数学表达 | 核心公式、变量解释、适用条件 |
| 直觉理解 | 例子、图示、常见误解 |
| 方法流程 | 流程图、步骤总结、适用场景 |
| 本节小结 | 一句话理解 + 重点回顾 |
| 讨论问题 | 适合继续思考的问题 |
